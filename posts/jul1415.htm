Everyday GIT

This is a collection of some git commands which I find useful in everyday tasks.

Add a remote

git remote add upstream <remote_name>

List all commits in branch, each in one line

git log --oneline —decorate

List all files in a commit

git ls-tree -r —name-only <commit>

List all files in a branch

git ls-tree -r —name-only <branch>

Compare a file in different branches

git diff <branch_1> <branch_2> <filename>

Draw git tree

git log --oneline --decorate --graph --all

Add <filename> to current branch from <branch>

git checkout <branch> <filename>

Create <branch_2> and copy content from <branch_1>

git checkout -b <branch_2> <branch_1>
